# dutodu

> **do it, dudu!** 오늘 할 일, 그냥 해버리자

todomate가 좋았는데, 내 작업 환경이랑 연결이 안 돼서 직접 만든 할일 관리 웹앱이에요.

---

## 이런 사람을 위해 만들었어요

- 할 일이 한꺼번에 떠올라서 머리가 복잡한 사람
- 취준하면서 기업별로 할 일을 깔끔하게 나누고 싶은 사람
- todomate처럼 예쁘고 간단한 걸 원하는 사람

---

## 주요 기능

**카테고리별 할일 관리**
- 취준, 사이드프로젝트, 일상 등 원하는 카테고리를 자유롭게 추가
- 기업별 서브카테고리 지원 (예: 취준 > 네이버 > 면접 준비)

**월간 캘린더**
- 날짜별 할일 확인
- 완료 현황 도트 표시

**시간 관리**
- 예상 소요시간 (15분, 1시간 등)
- 예정 시간 설정

**데이터 자동 저장**
- 체크할 때마다 자동 저장 (새로고침해도 안 날아감)

---

## 사용법

### 1. 실행
```bash
node todo-server.js
```
브라우저에서 `http://localhost:8080` 접속

### 2. 할일 추가
카테고리 옆 **+** 버튼 → 할일 입력 → 저장

### 3. 서브카테고리 추가
카테고리 하단 **＋ 서브카테고리 추가** 클릭 → 이름 입력

### 4. 할일 체크
동그라미 클릭하면 완료!

### 5. 수정/삭제
할일에 마우스 올리면 **···** 버튼 → 수정 또는 삭제

## 만든 이유

할 일이 쏟아지는 시기에, todomate로 관리하다가 느꼈어요.
"내가 작업하는 환경이랑 할일 앱이 따로 노는데?"

그래서 Claude Code랑 같이 쓸 수 있는 나만의 할일 앱을 만들었어요.
기업별 서브카테고리는, 취준할 때 매번 "○○기업 면접 준비"라고 풀네임 쓰기 귀찮아서 추가한 기능이에요.
